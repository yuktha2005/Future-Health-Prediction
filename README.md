# Future Health Prediction Portal (Work in Progress)
## Overview

A web app for predicting Diabetes and Heart Disease using trained AI/ML models.
Built with Flask, TensorFlow/Keras, and scikit-learn.

Note: This project is under development. Models may not be fully accurate.

## Features

Input personal and medical information.

Predict risk for Diabetes or Heart Disease.

Dark-themed UI (in progress).

Currently runs locally.
